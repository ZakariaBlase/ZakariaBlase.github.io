# Cyber-Workspace Browser

A zero-JS, file-explorer-style website to browse the [Cyber-Workspace](https://github.com/ZakariaBlase/Cyber-Workspace) OSINT & pentest tools collection online.

Click a category folder, then a tool — you're redirected straight to the tool/site. No install, no clone required.

Live at: `https://zakariablase.github.io/`

Generated statically from the source repo's `.url` shortcuts. To regenerate after updating the source collection, run `_generate-site.ps1` then `_generate-homepage.ps1` from the `Cyber-Workspace` repo.

For authorized security testing only.
